## 概述

本文档是从 ENS 英文官方文档翻译而来，其中的一些内容和细节得到了 ENS 官方团队的指导。文档中包含 “译注” 的地方是译者为了便于读者理解而进行的注释。

本地址是 ENS 中文文档的首发地址，也是目前唯一与官方文档保持同步的版本。包括登链社区在内的一些社区和个人，刊载了本文档。我们欢迎您转载，愿与您一同推动 ENS 的普及！

我们会持续更新本文档，保持与官方文档的同步。
